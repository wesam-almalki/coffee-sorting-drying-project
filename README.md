# Coffee Sorting AI System

## Overview
This project presents an AI-based system designed to improve coffee cherry sorting using computer vision techniques.

## Idea
The system analyzes visual features of coffee cherries to classify their quality automatically.

## Features
- Visual analysis using AI
- Quality classification concept
- Designed for real-time processing

## Technologies
- Computer Vision
- Machine Learning
- Python
